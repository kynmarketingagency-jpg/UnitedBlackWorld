### Navigation & Layout
- **Navbar.jsx** + **Navbar.module.css** - Main navigation bar with skull logo
- **Hero.jsx** + **Hero.module.css** - Homepage hero section

### Pages

### Styles
- **Admin.module.css** - Admin panel styles (skull design, pirate theme)
